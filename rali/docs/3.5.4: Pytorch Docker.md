# 3.5.4: Pytorch Docker
Steps to create docker for PyTorch training with MIVisionX RALI:

1.  Pull latest [rocm/pytorch docker](https://hub.docker.com/r/rocm/pytorch/tags) 
```
sudo docker pull rocm/pytorch:rocm4.0_ubuntu18.04_py3.6_pytorch
```

2. Install numactl
```
sudo apt-get install numactl
```

3. Install DLLOGGER
```
pip3.6 install git+https://github.com/NVIDIA/dllogger
```
4. Install torchvision
```
pip install --no-deps torchvision
```

6. Install apex
```
git clone --recursive https://github.com/ROCmSoftwarePlatform/apex
cd apex
python3.6 setup.py install --cuda_ext --cpp_ext
```

7. Clone MIVisionX. Run the setup script with '--neural_net off' option. 
```
git clone https://github.com/GPUOpen-ProfessionalCompute-Libraries/MIVisionX
cd MIVisionX
python MIVisionX-setup.py --neural_net off
```
8. Build and install MIVisionX. 
```
cd MIVisionX
mkdir build
cd build
cmake -DNEURAL_NET=OFF ../
make -j
sudo make install
```

9. Setup [rali_pybind](https://github.com/GPUOpen-ProfessionalCompute-Libraries/MIVisionX/tree/master/rali/rali_pybind#rali-python-binding)
```
sudo ./run.sh
```

10. Commit and save the new docker state
```
sudo docker commit <container ID> <new_image_name:tag_name>
sudo docker push <new_image_name:tag_name>
```
