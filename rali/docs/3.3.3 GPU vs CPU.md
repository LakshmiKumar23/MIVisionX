# 3.3.3 COnfiguring GPU vs CPU
Affinity can be set as CPU or GPU. We prefer to do the Data Loading on CPU, so that the GPU can be used for training. This would give the best performance 
Use the ```rali_cpu=True/False``` option to set affinity

```
class HybridTrainPipe(Pipeline):
	def __init__(self, batch_size, num_threads, device_id, data_dir, crop, rali_cpu = True):
		super(HybridTrainPipe, self).__init__(batch_size, num_threads, device_id, seed=12 + device_id,rali_cpu=rali_cpu)
```
