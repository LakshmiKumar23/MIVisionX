# 3.5.2 Tensoflow

An example of using RALI with Tensorflow for training. In this section, we show how to  convert the MNIST dataset to TFRecord data format, load the data, create a TF pipeline and, finally run the training. 

## 3.5.2.1 Pre-requisites 

* Install [TF1.15](https://www.tensorflow.org/install/pip)
* Build and install MIVisionX with option ```-DNEURAL_NET=OFF```
* Build and install rali_pybind plugin 
* Download and prepare MNIST tfrecord dataset

#### Option 1: Use a [Rocm Tensorflow docker](https://hub.docker.com/r/rocm/tensorflow) and follow the steps in this [document](https://github.com/LakshmiKumar23/MIVisionX/blob/lk/raliDocs/rali/docs/3.5.3%20Tensorflow%20Docker.md) to build your own docker.  
#### Option 2: Use docker under MIVisionX which has all the pre-requisites

## 3.5.2.2 Preparing the tfrecord dataset
Once the MNIST dataset is downloaded, [mnist_tfrecord.py script](https://github.com/GPUOpen-ProfessionalCompute-Libraries/MIVisionX/tree/master/rali/rali_pybind/example/tf_mnistTrainingExample) can be used to convert it to the required format.

```
def convert_to_tf_record(data_directory:str):
    """Convert the TF MNIST Dataset to TFRecord formats
    
    Args:
        data_directory: The directory where the TFRecord files should be stored
    """

    mnist = input_data.read_data_sets(
        data_directory, dtype=tf.uint8,
        reshape=False
    )
    
    convert_to(mnist.validation, 'validation', data_directory+'/val')

    convert_to(mnist.test, 'test', data_directory+'/test')
```

## 3.5.2.3 Data loading pipeline

* Imports for RALI

```
from amd.rali.plugin.tf import RALIIterator
from amd.rali.pipeline import Pipeline
import amd.rali.ops as ops
import amd.rali.types as types
```

* The usual RALI pipeline looks like the example below. It reads data from the TFRecords using ```TFRecordReader``` and uses ```ImageDecoderRaw``` to decode the raw images. 
The other required augmentation operations are defined in the Pipeline constructor. We then use ```define_graph``` to define the graph. 
```
#RALI pipeline
class HybridPipe(Pipeline):
	def __init__(self, feature_key_map, tfrecordreader_type, batch_size, num_threads, device_id, data_dir, crop, rali_cpu = True):
		super(HybridPipe, self).__init__(batch_size, num_threads, device_id, seed=12 + device_id,rali_cpu=rali_cpu)
		self.input = ops.TFRecordReader(path=data_dir, index_path = "", reader_type=tfrecordreader_type, user_feature_key_map=feature_key_map,
										features={
											'image/encoded':tf.FixedLenFeature((), tf.string, ""),
											'image/class/label':tf.FixedLenFeature([1], tf.int64,  -1),
											'image/filename':tf.FixedLenFeature((), tf.string, "")
										},
										)
		rali_device = 'cpu' if rali_cpu else 'gpu'
		decoder_device = 'cpu' if rali_cpu else 'mixed'
		self.decode = ops.ImageDecoderRaw(user_feature_key_map=feature_key_map, device=decoder_device, output_type=types.RGB)
		#self.res = ops.Resize(device=rali_device, resize_x=crop[0], resize_y=crop[1])
		self.cmnp = ops.CropMirrorNormalize(device="cpu",
											output_dtype=types.FLOAT,
											output_layout=types.NCHW,
											crop=(crop,crop),
											image_type=types.GRAY,
											mean=[0 ,0,0],
											std=[255,255,255], mirror=0)
		self.coin = ops.CoinFlip(probability=0.5)
		print('rali "{0}" variant'.format(rali_device))

	def define_graph(self):
		inputs = self.input(name ="Reader")
		images = inputs["image/encoded"]
		labels = inputs["image/class/label"]
		images = self.decode(images)
		#rng = self.coin()
		output = self.cmnp(images)
		return [output, labels]
```
